# Linux File System

## Common Directories
/       &nbsp;&nbsp; - root directory<br>
/home   &nbsp;&nbsp; - user home directories<br>
/etc    &nbsp;&nbsp; - configuration files<br>
/var    &nbsp;&nbsp; - logs and variable files<br>
/bin    &nbsp;&nbsp; - essential binaries<br>
/usr    &nbsp;&nbsp; - user installed software<br>

## Navigation Commands
pwd     &nbsp;&nbsp; # show current directory<br>
ls      &nbsp;&nbsp; # list directory contents<br>
cd /etc &nbsp;&nbsp; # change directory<br>
