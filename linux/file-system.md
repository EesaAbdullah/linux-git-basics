# Linux File System

## Common Directories
/        - root directory<br>
/home    - user home directories<br>
/etc     - configuration files<br>
/var     - logs and variable files<br>
/bin     - essential binaries<br>
/usr     - user installed software<br>

## Navigation Commands
pwd      # show current directory<br>
ls       # list directory contents<br>
cd /etc  # change directory<br>
